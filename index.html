Introducción

Bienvenido a la sección de desarrolladores de 'The Pintxopo'. En ésta sección encontrarás la documentación necesaria para hacer uso de la API pública de nuestra aplicación. En resumen, te mostraremos como acceder a la información de los Pintxos y los Bares y obtenerla en forma de paquetes JSON.

Información de los recursos

Formato de respuesta: JSON

Requiere estar logeado? No

Uso límitado? No

Pintxos

Para obtener la información necesaria sobre los pintxos de la aplicación. Podrás obtner tanto la lista de pintxos, un pintxo por nombre como parámetro o el pintxo pasándole un número como parámetro que te devolverá el pintxo que corresponde en la lista.

GET Pintxos

Para obtener la lista de los pintxos que hay en la base de datos usaremos la siguiente URL:

http://www.thepintxopo.com/pintxosDatos

Esto nos devolverá un array con un JSON en el que se incluyen los nombres y las descripciones de cada pintxo.

Ejemplo:

[{"nombre":"Maki de atún","descripcion":"Delicioso bocado de atún envuelto en arroz sushi-meshi y alga nori."},{"nombre":"Bola de carne","descripcion":"Uno de los pintxos más habituales, un bocado de carne con un toque picante"}]

GET Pintxos:nombre

Para obtener un pintxo introduciendo su nombre usaremos la siguiente URL:

http://www.thepintxopo.com/pintxosDatos/nombre/:nombre

Siendo /:nombre un parámetro que tendremos que introducir, ej: 'Maki de atún'. Esto nos devolverá un JSON en el que se incluyen el nombre y la descripción de ese pintxo.

Ejemplo:

{"nombre":"Maki de atún","descripcion":"Delicioso bocado de atún envuelto en arroz sushi-meshi y alga nori."}

GET Pintxos:número

Para obtener un pintxo introduciendo su número de orden usaremos la siguiente URL:

http://www.thepintxopo.com/pintxosDatos/:numero

Siendo /:numero un parámetro que tendremos que introducir, ej: '0'. Esto nos devolverá un JSON en el que se incluye el nombre de ese pintxo.

Ejemplo:

Maki de atún

Bares

Para obtener la información necesaria sobre los bares de la aplicación. Podrás obtner tanto la lista de bares, un bar por nombre como parámetro o el bar pasándole un número como parámetro que te devolverá el bar que corresponde en la lista.

GET Bares

Para obtener la lista de los bares que hay en la base de datos usaremos la siguiente URL:

http://www.thepintxopo.com/baresDatos

Esto nos devolverá un array con un JSON en el que se incluyen el nombre y las coordenadas de cada bar.

Ejemplo:

[{"coordX":"43.317020","coordY":"-1.971988","nombre":"Bar Jesús"},{"coordX":"43.326289","coordY":"-1.971695","nombre":"Elosta"}]

GET Bares:nombre

Para obtener un bar introduciendo su nombre usaremos la siguiente URL:

http://www.thepintxopo.com/pintxosDatos/nombre/:nombre

Siendo /:nombre un parámetro que tendremos que introducir, ej: 'Bar Jesús'. Esto nos devolverá un JSON en el que se incluyen el nombre y las coordenadas de ese bar.

Ejemplo:

{"coordX":"43.317020","coordY":"-1.971988","nombre":"Bar Jesús"}

GET Bares:número

Para obtener un bar introduciendo su número de orden usaremos la siguiente URL:

http://www.thepintxopo.com/pintxosDatos/:numero

Siendo /:numero un parámetro que tendremos que introducir, ej: '0'. Esto nos devolverá un JSON en el que se incluye el nombre de ese bar.

Ejemplo:

Bar Jesús
